# limehq/homebrew-tap

Homebrew tap for [Munkel](https://github.com/limehq/munkel) — ephemeral
whispers that slide out of the MacBook notch.

## Install

```sh
brew install limehq/tap/munkel
```

Installs `Munkel.app` into `/Applications` and the `munkel` CLI into
`$(brew --prefix)/bin`. Casks here are updated automatically by the
[release workflow](https://github.com/limehq/munkel/blob/main/.github/workflows/release.yml).
